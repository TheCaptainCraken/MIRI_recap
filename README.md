# MIRI Advanced Computing | Preparation Material

Self-study notes for incoming **MIRI** students at **UPC** (Universitat Politècnica de Catalunya).

## What this is

A preparation handbook covering the math and CS background you need going into the first semester of the *Advanced Computing* specialization. Topics span **Linear Algebra, Calculus, Probability, and Concurrency** — the goal is to get you ready to succeed from day one.

## How to get it

**Download the PDF (recommended)**
Grab the latest build from the [Releases page](https://github.com/TheCaptainCraken/MIRI_recap/releases).

**Option 2 — Compile it yourself.**
You'll need [Typst](https://github.com/typst/typst) installed. Then, from the repo root:

```sh
typst compile --font-path fonts main.typ miri_preparation.pdf
```

This produces `miri_preparation.pdf`.
